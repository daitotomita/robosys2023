# robosys2023

#インストール方法
*  自身のpythonの動く環境に導入する。

#使い方
*  任意の数字をスペース区切りでいくつか入力すると、その入力した数を足してくれます。
*  以下のように入力することで1-10の全ての数を足してくれます。

***
$ seq 10 | ./plus_stdin
50
***
